# 🚀 بوت‌کمپ هوش مصنوعی مولد - پاییز 1404

این مخزن شامل تمرین‌ها و تسک‌های کدنویسی پایتون مربوط به دوره‌های مختلف **بوت‌کمپ هوش مصنوعی مولد آکادمی همراه اول** در پاییز 1404 است.  
هدف از این مخزن، مستندسازی مسیر یادگیری و اشتراک‌گذاری پروژه‌ها و تمرین‌ها با جامعه‌ی توسعه‌دهندگان و علاقه‌مندان به هوش مصنوعی مولد است.

## 🎯 اهداف مخزن
- جمع‌آوری و سازمان‌دهی تسک‌های هر دوره
- نمایش روند یادگیری و پیشرفت در حوزه‌ی هوش مصنوعی مولد
- ایجاد مرجع آموزشی برای علاقه‌مندان به پایتون و هوش مصنوعی
